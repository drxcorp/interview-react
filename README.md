# Modern E-commerce Store

A modern, minimal e-commerce website built with React 19 for senior developer interviews.

## Getting Started

```bash
npm install
npm run dev
```

## Tech Stack

- React 19.0
- TypeScript
- Vite
- Zustand (State Management)

## Features

- Product listing with filtering and sorting
- Product detail modal
- Shopping cart with quantity management
- Checkout flow with validation
- Responsive design
- Modern minimal UI

## Project Structure

```
src/
├── components/     # React components
├── data/          # Mock data
├── store/         # Zustand store
├── App.tsx        # Main app component
└── main.tsx       # Entry point
```
